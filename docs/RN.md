RN-01: El sistema solo permitirá el registro de usuarios que utilicen una dirección de
correo electrónico con el dominio oficial de la institución (@upchiapas.edu.mx).
Cualquier otro dominio será rechazado automáticamente.
● RN-02: No se permitirán registros duplicados. El sistema validará que el correo
electrónico no exista en la base de datos antes de proceder con la creación de una
nueva cuenta.
● RN-03: Para que una publicación de apuntes sea exitosa, el usuario debe completar
obligatoriamente los siguientes campos: Título, Cuatrimestre, Asignatura y al menos un
archivo adjunto (imagen, documento PDF o texto).
● RN-04: La visualización de apuntes es pública para todos los usuarios, pero las
acciones de interactuar (reaccionar), comentar y publicar nuevos apuntes requerirán
estrictamente que el usuario haya iniciado sesión
● RN-05: El sistema contará con un filtro de palabras prohibidas. Cualquier comentario
que incluya lenguaje vulgar, ofensivo o enlaces detectados como spam será bloqueado
automáticamente antes de publicarse o eliminado por el sistema.
● RN-06: Queda estrictamente prohibida la publicación de material de evaluación directa
(exámenes vigentes, respuestas de tareas completas o contenido vulgar). Las
publicaciones que infrinjan esta norma serán eliminadas de forma definitiva por el
sistema o los administradores.
● RN-07: Cuando una publicación o comentario acumule tres (3) reportes por parte de la
comunidad, el sistema la ocultará automáticamente de la vista pública (estado: inactivo)
y generará una alerta en el panel de administración para su revisión y dictamen final.
● RN-08: Un estudiante puede eliminar sus propios apuntes publicados en cualquier
momento, lo cual borrará también de forma permanente todos los comentarios y
reacciones asociados a esa publicación.
● RN-09: Si un estudiante acumula tres publicaciones o comentarios dados de baja por
infringir las normas (RN-05 o RN-06), su cuenta será suspendida temporalmente por 5
días; a la reincidencia, la baja será definitiva.
Nota: esta regla no se aplicara por el momento
● RN-10: Para garantizar la velocidad de la plataforma, el tamaño máximo permitido por
archivo adjunto (PDF, imágenes) será de 20 MB. Si un archivo supera este límite, el
sistema bloqueará la subida y sugerirá al usuario comprimirlo o subir un enlace externo
(como Google Drive).
● RN-11: El sistema solo aceptará archivos con extensiones seguras y legibles, limitadas
a: .pdf, .png, .jpg Y .C y Cualquier otro formato (como ejecutables .exe o archivos
extraños) será rechazado por motivos de seguridad.
